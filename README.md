## Getting Started

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

Edit the homepage in `app/page.tsx` and global layout in `app/layout.tsx`.

## Deploy on Vercel

Push to the `main` branch of this repository, then import the repo in Vercel and follow the prompts. The default Next.js settings work out of the box.
